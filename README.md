# Models Viewer | 模型查看器

[English](#english) | [中文](#中文)

---

## English

A Vue 3-based web application for viewing and comparing AI model information across different providers. This application helps developers explore available models, their capabilities, costs, and technical specifications in a unified interface.

**Live Demo**: https://models.anya2a.com/

**Data Source**: https://github.com/ThinkInAIXYZ/PublicProviderConf

## Features

- **Comprehensive Model Catalog**: View models from multiple AI providers in one place
- **Advanced Filtering**: Search by model ID, name, or filter by provider
- **Capability Comparison**: Compare model features like reasoning, tool calls, multimodal support
- **Cost Analysis**: View input/output costs for different models
- **Limit Information**: Check context and output limits
- **Virtual Scrolling**: Efficient handling of large datasets with virtual scrolling

## Development Setup

### Install Dependencies

```sh
pnpm install
```

### Development Server

```sh
pnpm dev
```

### Build for Production

```sh
pnpm build
```

## 中文

基于 Vue 3 的网络应用程序，用于查看和比较不同提供商的 AI 模型信息。该应用程序帮助开发者在统一的界面中探索可用模型、功能、成本和技术规格。

**在线演示**: https://models.anya2a.com/

**数据来源**: https://github.com/ThinkInAIXYZ/PublicProviderConf

## 功能特性

- **全面的模型目录**：在一个地方查看来自多个 AI 提供商的模型
- **高级过滤**：按模型 ID、名称搜索或按提供商过滤
- **功能比较**：比较模型特性，如推理、工具调用、多模态支持
- **成本分析**：查看不同模型的输入/输出成本
- **限制信息**：检查上下文和输出限制
- **虚拟滚动**：使用虚拟滚动高效处理大型数据集
